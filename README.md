# Pomodoro Light

A minimalist Pomodoro timer tool for the Light Phone III, built with the LightOS SDK.

## Features
- Focus/Break cycle timer (15-60 min focus, 5-30 min break, in 5-minute steps)
- Circular progress ring visualization
- Daily stats (completed pomodoros, total focus minutes)
- Minimalist design following LightOS UI components

## Built with
- Kotlin + Jetpack Compose
- LightOS SDK (com.thelightphone.sdk)

---

This project is a fork of [lightphone/light-sdk](https://github.com/lightphone/light-sdk), 
the official scaffold for building tools for the Light Phone III.
